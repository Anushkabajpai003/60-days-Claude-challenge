# Day 48 – Compare & Decide Builder

## Overview

Today I built **Compare & Decide Builder**, an AI-powered web application that transforms complex decisions into structured, evidence-based comparisons.

Instead of relying on opinions or intuition, the application researches real-world information, compares multiple options across measurable criteria, and provides transparent rankings backed by verifiable sources.

---

## Problem Statement

Choosing between products, services, careers, universities, or software often requires gathering information from multiple websites.

Most comparison tools:

* Use static rankings.
* Hide their evaluation process.
* Don't allow personalized priorities.
* Rarely provide transparent citations.

This project addresses these limitations by creating a comparison system that is interactive, explainable, and research-driven.

---

## Features

* Interactive interview flow using multiple-choice questions.
* Compares multiple options using measurable criteria.
* Researches and verifies information from real sources.
* Displays complete source citations.
* Allows users to assign custom weights to each criterion.
* Updates rankings instantly based on user priorities.
* Clearly marks estimated or placeholder values.
* Includes a detailed "How this was researched" section.
* Responsive and modern user interface.
* Built using pure HTML, CSS, and JavaScript.

---

## Workflow

1. User selects what they want to compare.
2. The application identifies comparison criteria.
3. Relevant information is collected from trusted sources.
4. Data is validated and organized.
5. Users adjust the importance of each criterion.
6. Scores are recalculated in real time.
7. Final rankings are displayed with full transparency.
