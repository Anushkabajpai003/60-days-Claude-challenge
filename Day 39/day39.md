# Day 39 – PDF Splitter & Merger

## 📄 Project Overview

**PDF Splitter & Merger** is a premium single-page web application that enables users to split and merge PDF documents directly in the browser. All processing happens locally on the user's device, ensuring complete privacy and offline functionality after the initial load.

The application features a modern, responsive interface with intuitive workflows, drag-and-drop support, live previews, and accessibility-focused interactions.

---

## ✨ Features

### 📑 PDF Splitter

* Upload PDF documents
* Automatic page count detection
* Visual page thumbnails
* Document preview
* Split by custom page ranges
* Split after specific pages
* Split every **N** pages
* Extract selected pages
* Multiple split ranges in one operation
* Real-time page range validation
* Output preview before processing

### 📚 PDF Merger

* Drag-and-drop PDF upload
* Multiple file selection
* Visual file previews
* Display page counts
* Reorder files using drag-and-drop
* Total file and page statistics
* Generate merged PDF
* Download merged document
