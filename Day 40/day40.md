# Day 40 – AI Assistant Builder 🤖

## Overview
AI Assistant Builder is an interactive web application that helps users design and generate a production-ready AI assistant through a guided interview process.

Instead of asking users to write long prompts manually, the application collects requirements step by step and automatically creates:

- A professional system prompt
- A premium user interface
- Documentation explaining the design

The goal is to simplify AI assistant creation while following good product design and prompt engineering practices.

- ✅ Interactive MCQ-based interview

## Interview Flow

### Step 1
Choose the assistant domain and niche.

### Step 2
Define the target audience and primary outcome.

### Step 3
Select user input types:
- Free text
- Uploaded file
- Form fields
- Multi-turn conversation
- Documents

### Step 4
Choose desired outputs:
- Reports
- Recommendations
- Scores
- Generated documents
- Conversational responses

### Step 5
Choose the assistant personality:
- Professional
- Friendly
- Expert
- Playful
