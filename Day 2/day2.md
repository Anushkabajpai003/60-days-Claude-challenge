# Day 2 - Prompt Engineering

## Content of the Day

Today's topic was **Prompt Engineering**.

Prompt Engineering is the process of writing clear and detailed instructions for AI tools like Claude to get better results. The quality of the output depends heavily on the quality of the prompt. By providing context, audience, format, tone, and objectives, users can significantly improve AI-generated responses.

### Improved Prompt

**Prompt:**

## Key Learnings

1. Better prompts produce better outputs.
2. Adding context helps AI understand the task more accurately.
3. Specifying audience and tone improves relevance.
4. Clear formatting instructions create more structured responses.
5. Prompt Engineering is a key skill for effectively using AI tools like Claude.

---

## Conclusion

Prompt Engineering is not just about asking questions; it is about giving clear instructions. Small improvements in prompts can lead to significantly better AI-generated results.
