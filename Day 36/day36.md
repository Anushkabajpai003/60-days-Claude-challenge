# Cognitive Pattern Explorer

## Project Overview

**Cognitive Pattern Explorer** is a psychology-inspired self-reflection web application designed to help users explore different thinking tendencies through interactive scenarios, priority ranking exercises, timeline mapping, and personalized reflections.

The experience is educational in nature and encourages self-awareness through exploration rather than evaluation. The application avoids diagnosis or assessment and instead focuses on reflective learning.

---

## Technologies Used

* HTML5
* CSS3
* Vanilla JavaScript
* Native Drag and Drop API
* Responsive Design
* Offline-First Architecture

---

## Features

### 1. Start Screen

* Calm Mode
* Stress Mode
* Guided onboarding experience

### 2. Discover Your Thinking Style

Interactive scenario-based choices exploring:

* Analytical Thinker
* Emotional Intuitive
* Overthinking Loop Style
* Action-First Decision Maker
* Balanced Reflective Thinker

### 3. Priority Explorer

Drag-and-drop activity allowing users to rank personal priorities and values.

### 4. Thinking Timeline Mapper

Interactive timeline builder where users arrange thought processes and decision-making sequences.

### 5. Reflection Journal

Personalized reflection area including:

* Thinking tendency breakdown
* Insight summary
* Reflection prompts
* Growth-oriented observations

# Key Learnings

### User Experience Design

Creating a self-reflection tool requires careful language choices to ensure users feel supported rather than judged.

### Interactive Learning

Gamified interactions such as drag-and-drop activities significantly improve engagement and participation.

### Psychology-Inspired Design

Reflective experiences work best when they encourage curiosity instead of assigning rigid categories.

### Accessibility

Keyboard navigation, responsive layouts, and reduced-motion support help create a more inclusive experience.

### Frontend Development

Building reusable drag-and-drop components in Vanilla JavaScript reinforces modular thinking and maintainable code design.

---

# Project Outcome

This project demonstrates how interactive web applications can transform self-reflection into an engaging and exploratory experience while maintaining accessibility, responsiveness, and complete offline functionality.

Educational Use Only — This project does not provide psychological assessment, diagnosis, or clinical evaluation.
